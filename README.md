# pocket_health_challenge
A simple microservice to upload DCM files, read DCM/PNG files, fetch DCM attributes by tag

## APIs
1. 'upload_file()'
2. 'download_dicom_file()'
3. 'download_image_file()'
4. 'get_header_attributes()'
